# Package Review Checklist

- [ ] Package builds with `colcon build`.
- [ ] Tests pass with `colcon test`.
- [ ] README is complete.
- [ ] Topics and parameters are documented.
- [ ] No hard-coded local paths.
- [ ] No secrets or credentials.
- [ ] Simulation mode supported where practical.
- [ ] Rosbag replay considered where applicable.
- [ ] Certification impact documented if relevant.
